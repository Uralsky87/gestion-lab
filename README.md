# Gestión Lab (PWA)

PWA mobile-first para control personal de producción. No requiere backend.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`

## Estructura

- `src/components` UI reutilizable (header, tabs)
- `src/layouts` layout principal
- `src/pages` secciones principales
- `src/routes` rutas y navegación

## PWA

- Manifest y service worker configurados con `vite-plugin-pwa`.
- Ícono principal en `public/icon.svg`.
